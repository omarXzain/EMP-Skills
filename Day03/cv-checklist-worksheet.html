قائمة تحسين السيرة الذاتية - CV Checklist

الجلسة 03
الاسم:
التخصص:
التعليمات: راجع سيرتك الذاتية الحالية (أو خطط لواحدة) مقابل هذه القائمة. ضع علامة على العناصر الموجودة لديك، واستخدم خطة التحسين في الأسفل لكتابة 3 تغييرات محددة ستقوم بها هذا الأسبوع.
1 الترويسة / معلومات الاتصال - Header
  • الاسم الكامل معروض بشكل بارز
  • إيميل مهني (ليس اسم دلع أو غير مناسب)
  • رقم الهاتف مع رمز الدولة (+962)
  • رابط LinkedIn مضمّن
  • رابط Portfolio / GitHub / Behance (إن وُجد)
  • لا توجد معلومات شخصية غير ضرورية (العمر، الحالة الاجتماعية، الصورة*)
2 الملخص المهني - Professional Summary
  • 3-4 أسطر كحد أقصى
  • يوضح من أنت وهدفك المهني
  • يبرز أهم المهارات المتعلقة بالوظيفة
  • مخصص لكل وظيفة (ليس عام)
اكتب مسودة ملخصك المهني:
3 قسم المهارات - Skills Section
  • يتضمن مهارات تقنية (Hard Skills) ومهارات ناعمة (Soft Skills)
  • يستخدم كلمات مفتاحية من إعلان الوظيفة المستهدف
  • المهارات محددة (مثل: "React, Node.js" وليس فقط "programming")
  • لا توجد مهارات قديمة أو غير ذات صلة
4 التعليم - Education
  • الشهادة، الجامعة، وسنة التخرج مذكورة
  • المعدل مضمّن إذا كان قوي (3.5+)
  • مواد دراسية أو مشروع تخرج ذو صلة مذكور
  • الشهادات المهنية مدرجة (Google, HubSpot, Meta, إلخ)
5 المشاريع / الأعمال - Projects / Portfolio
  • على الأقل 2-3 مشاريع حقيقية موصوفة
  • كل مشروع له نتائج قابلة للقياس (أرقام، نتائج)
  • التقنيات أو الأدوات المستخدمة مذكورة لكل مشروع
  • روابط للمشاريع المباشرة، Repos، أو أعمال Portfolio
6 الخبرة - Experience
  • يتضمن تدريب، عمل حر، عمل جزئي، أو عمل تطوعي
  • يستخدم أفعال عمل: "Built", "Led", "Designed", "Increased"
  • يتضمن مقاييس حيثما أمكن: "Increased X by Y%"
  • الخبرة الأحدث مدرجة أولاً (ترتيب زمني عكسي)
+ متطلبات خاصة بالتخصص

التسويق الرقمي - Digital Marketing

  • اذكر المنصات: Google Ads, Meta Ads, SEO tools
  • أظهر نتائج الحملات: CTR, ROAS, reach
  • رابط لنماذج محتوى
  • الشهادات: Google, HubSpot, Meta Blueprint

تطوير البرمجيات - Software Development

  • رابط GitHub إجباري - اجعل الـ repos نظيفة
  • اذكر اللغات والأطر (Frameworks)
  • وصف: مشكلة ← حل ← تقنيات
  • ضمّن تطبيقات deployed / روابط مباشرة

تطبيقات الموبايل - Mobile Application

  • اذكر المنصات: iOS (Swift), Android (Kotlin), Flutter
  • ضمّن روابط متجر التطبيقات إن نُشرت
  • أظهر حساسيتك لـ UI/UX
  • تكاملات API وخبرة Backend

التصميم الجرافيكي - Graphic Design

  • رابط Portfolio (Behance/Dribbble) في الأعلى
  • الأدوات: Adobe CC, Figma, Canva, Illustrator
  • الأنواع: branding, UI, print, social media
  • أظهر العملية، ليس فقط النتيجة النهائية
